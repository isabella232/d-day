Новые возможности серверной ГИС NextGIS Web

NextGIS Web - наш основной продукт, выступающий в качестве ядра корпоративной ГИС. NextGIS Web выступает хранилищем данных, обеспечивает настройку прав доступа, предоставляет доступ к данным по протоколам WMS/WFS и API и многое другое. В этой демонстрации мы покажем что существенно изменилось за прошедший год. Из основного, больше нет необходимости настраивать символику отдельно для настольных и веб-приложений, работа со стилями стала значительно проще. NextGIS Web также получил возможность отдавать клиенту большие объёмы векторных данных с помощью технологии векторных тайлов. Это радикально ускорит работу с векторными геоданными в ваших приложениях.

NextGIS Борщ — новая система сборки ПО ГИС и сопутствующие сервисы

Основой любого софта ( не важно, настольного, веб или мобильного) являются библиотеки. Их использование позволяет разделять сложные программы на логически связанные части, легче отлаживать ошибки, обновлять компоненты.

Однако собирать, подключать к своим программам, поддерживать в актуальном состоянии ГИС-библиотеки довольно сложно. При обновлении одной из библиотек надо пересобирать все библиотеки, которые от нее зависят. Также для различных программ нужны по-разному собранные библиотеки. Мы разработали систему сборки ГИС-библиотек, которая упрощает процессы сборки и обновления программ и библиотек: достаточно буквально 3 строчек кода в скрипте CMake.
